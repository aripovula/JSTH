App summary:


In developing the app's 'back end' followings were used:

      Java 8 ( Java 8 Stream was used as often as practicable )
      Spring BOOT ( 1.5.8 )
      Spring MVC
      Spring Session
      Spring Security
      JPA, Hibernate
      MySQL
      Google gson ( for generating data in JSON format )
      Maven
      Eclipse Oxygen

The spring.datasource.tomcat.connection is used to ensure that any character is properly stored using UTF-8 encoding

In developing the app's 'front end' followings were used:

      Thymeleaf
      Bootstrap 3
      Bootstrap smart menus ( responsive menu )
      Javascript, including JSON parsing and dynamic creation and binding of DOM elements
      JQuery was used for animations
      Googles' Code Prettify library ( https://github.com/google/code-prettify ) was used to display any source code, flashcards and example code
      Other resources / libraries were used to create 'Personal info' ( resumé ) page and source code viewer with resizable split screens

This app is NOT a copy / paste of an existing application. It was developed by me ( I deployed Spring BOOT app and added all classes and templates myself ).
